## Stack technique

| Catégorie | Technologies |
|-----------|-------------|
| **Langages** | HTML5, CSS3 (Grid, Custom Properties, `color-mix`), JavaScript ES6+ |
| **Mise en page** | CSS Grid — Bento Grid layout (4 colonnes) |
| **Fonts** | Inter (Google Fonts) |
| **Animations** | IntersectionObserver, transitions CSS, keyframes |
| **Thème** | Dark / Light — toggle stocké dans `localStorage` (`pref-theme-v2`) |
| **Accessibilité** | ARIA labels, focus trap modal, `prefers-reduced-motion` |
| **Build** | Aucun — fichiers statiques, zéro dépendance |
